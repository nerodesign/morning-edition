# Morning Edition

Daily editorial magazine of trending AI prompts, templates, and agent patterns — curated from X and Reddit, published at 7am AWST.

**[Read Today's Issue →](https://nerodesign.github.io/morning-edition/)**

## What's Inside

Every morning, 10 trending prompts and templates across:
- **Agent Templates** — system prompts, SOUL.md patterns, agent architectures
- **Image Generation** — Midjourney SREF codes, cinematic formulas, conceptual templates
- **AI Video** — filmmaker-style structured prompts, JSON workflows
- **Skills & Tools** — Claude skills, agent patterns, prompt engineering techniques

## Design

Editorial magazine layout with 10 distinct spreads per issue — Hero, Midnight, Rose Stamp, Terminal, Academic, Manifesto, Editorial, Big Stat, Brutalist, and Finale. Fraunces + Inter typography.

## How It Works

A cron job runs daily at 7am AWST, scans X and Reddit for trending content, curates the top 10, and publishes to this repo via GitHub Pages.

Built by [StudioOps](https://github.com/nerodesign).
